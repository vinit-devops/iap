## Implemented

Created [`ROADMAP.yaml`](../../ROADMAP.yaml): all 20 phases from the master roadmap (0, 0.5, 1–18) with per-phase priority (P0–P4 per roadmap §11), dependency edges (`dependsOn`), milestone decomposition, status tracking (`pending | in-progress | completed | blocked`), evidence lists (file paths), and verbatim exit criteria. Phase completion is gated on exit criteria, never dates (roadmap §17 closing rule). The executing agent updates status + evidence after every milestone; each milestone additionally produces a reviewable document under `docs/milestones/`.
