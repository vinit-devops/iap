## Real bugs caught by live execution (mock tests could not)

The executor was mock-verified before this run; live AWS surfaced two genuine defects, both
fixed and now regression-tested (`packages/deploy-aws/test/create.test.ts`):

1. **`Unknown Attribute FifoQueue.`** — the SQS handler sent `FifoQueue=false` for a standard
   queue; AWS only accepts `FifoQueue=true` (omit otherwise). Fixed: omit `FifoQueue` unless true.
2. **`Invalid value for the parameter MessageRetentionPeriod.`** — the AWS mapping emits an IaP
   duration (`7d`), but SQS wants an integer count of seconds (`604800`). Fixed: coerce IaP
   durations → seconds for second-valued SQS attributes; projection normalized on both desired
   and live sides so idempotence (no-op) compares equal.

A CLI wiring bug was also found and fixed: `--profile` collided with the IaP merge-profile, so
AWS credentials now use `--aws-profile` (`--profile` stays the IaP profile).
